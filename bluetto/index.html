隐私政策

欢迎使用 蓝牙调试助手-SPP(串口)通信（以下简称“本应用”）。 本应用重视用户的隐私与个人信息保护。本隐私政策旨在说明： 我们如何收集、使用、存储以及保护您的信息。

一、应用基本信息

  • 应用名称：蓝牙调试助手-SPP(串口)通信
  • 应用包名：com.zzf.bluetoothsmp
  • 应用类型：蓝牙串口（SPP）通信与调试工具

二、权限及收集信息说明

1. 蓝牙权限

本应用需要访问设备的蓝牙权限，以便搜索、连接和管理蓝牙串口（SPP）设备。 该权限仅用于本地蓝牙通信功能，不会上传或共享用户数据。

2. 广告 ID（AD_ID）及分析

本应用使用 Firebase Analytics 收集设备的广告 ID（AD_ID）以及应用使用情况， 仅用于统计应用性能和功能优化。 收集的广告 ID 不会用于个性化广告，也不会与您的个人身份信息关联。

3. 崩溃日志

本应用使用 Firebase Crashlytics 收集应用崩溃日志，以提升应用稳定性。 收集内容仅限崩溃信息，不包含个人身份信息。

三、第三方服务

本应用使用以下第三方服务：

  • Firebase Analytics：收集 AD_ID 及应用使用情况，仅用于分析和优化功能。
  • Firebase Crashlytics：收集崩溃日志，用于提升应用稳定性。

本应用不会将用户数据共享给广告网络，也不会用于个性化广告。

四、未成年人隐私保护

本应用不面向未成年人提供专属服务，也不会主动收集任何未成年人的个人信息。

五、数据安全

本应用不会上传或存储用户个人数据到开发者服务器。 所有操作均在用户设备本地进行，收集的数据仅用于分析和崩溃日志。

六、隐私政策更新

如果本隐私政策发生变更，我们将在本页面更新内容。 更新后的隐私政策一经发布即生效。
